# Slides archivadas

Slides que salieron del deck pero se conservan por si se quiere volver atrás.
El build (`scripts/build.py`) solo arma `presentacion/` y `moldes/`, así que **nada de esta
carpeta entra al deck**.

## Contenido

- **`34-reco-no-escalar.html`** — versión previa de la slide de recomendación, con el gatillo de
  inversión expresado como **hogares activos (100-110)**. Reemplazada el 2026-06-29 por
  `presentacion/34-reco-costo-oportunidad.html`, que reencuadra el gatillo en el **costo de
  oportunidad de Ricardo** (más fiel al Informe N°3).

## Cómo volver atrás

Mover el archivo de vuelta a `presentacion/` (renombrándolo según la posición deseada) y correr
`python3 scripts/build.py`.
